# homebrew-idx

> Homebrew tap for [idx](https://github.com/eltu/idx) — a fast local file indexer and search tool.

## What is idx?

**idx** is a blazing-fast CLI tool for indexing and searching files on your local machine. It is designed to be lightweight, efficient, and easy to use from the terminal.

## Installation

Add this tap and install `idx` with a single command:

```sh
brew tap eltu/idx
brew install idx
```

## Usage

Once installed, you can start using `idx` right away:

```sh
idx --version
```

For full documentation and usage examples, visit the main repository:
**[github.com/eltu/idx](https://github.com/eltu/idx)**

## Updating

To update `idx` to the latest version:

```sh
brew update
brew upgrade idx
```
